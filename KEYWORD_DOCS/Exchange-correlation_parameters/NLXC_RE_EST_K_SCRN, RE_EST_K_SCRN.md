# NLXC\_RE\_EST\_K\_SCRN or RE\_EST\_K\_SCRN (.param)

## Keyword type

Logical

## Description

This keyword determines whether or not the estimate of the Thomas-Fermi screening length in the screened exchange formalism is updated as
the density changes. NLXC\_RE\_EST\_K\_SCRN is the latest form of this keyword, while the obsolete version, RE\_EST\_K\_SCRN, is supported for
backward compatibility.

## Default

FALSE

## Example

```

NLXC_RE_EST_K_SCRN : TRUE
```
