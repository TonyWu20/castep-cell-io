# PHONON\_CONVERGENCE\_WIN (.param)

## Keyword type

Integer

## Description

This keyword determines the size of the convergence window during a phonon calculation.

The linear response convergence criteria must be met for PHONON\_CONVERGENCE\_WIN iterations before acceptance.

## Default

2

## Example

```

PHONON_CONVERGENCE_WIN : 4
```
