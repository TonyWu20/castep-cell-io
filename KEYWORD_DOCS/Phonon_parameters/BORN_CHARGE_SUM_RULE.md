# BORN\_CHARGE\_SUM\_RULE (.param)

## Keyword type

Logical

## Description

This keyword specifies whether to correct the Born effective charge tensor explicitly to enforce the sum rule that effective charges sum to
zero.

## Default

FALSE

## Example

```

BORN_CHARGE_SUM_RULE : TRUE
```
