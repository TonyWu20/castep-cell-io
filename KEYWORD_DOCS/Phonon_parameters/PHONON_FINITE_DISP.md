# PHONON\_FINITE\_DISP (.param)

## Keyword type

Real

## Description

This keyword specifies the amplitude of the ionic perturbation that will be used in finite displacement phonon calculations.

## Default

0.01 Bohr

## Example

```

PHONON_FINITE_DISP : 0.01 ANG
```
