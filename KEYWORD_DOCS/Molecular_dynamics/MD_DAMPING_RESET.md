# MD\_DAMPING\_RESET (.param)

## Keyword type

Integer

## Description

This keyword specifies the number of molecular dynamics steps between recalculations of the damping parameters. If this parameter is set to
0, the damping parameters remain fixed.

## Default

30

## Example

```

MD_DAMPING_RESET : 20
```
