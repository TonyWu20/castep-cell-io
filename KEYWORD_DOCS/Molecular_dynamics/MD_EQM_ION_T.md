# MD\_EQM\_ION\_T (.param)

## Keyword type

Real

## Description

This keyword is used to set the MD barostat parameter for use in enhanced MD equilibration method (when
[MD\_EQM\_METHOD](k_md_eqm_method_castep.htm) is not set to NONE).

## Default

By default this has the same value as [MD\_ION\_T](k_md_ion_t_castep.htm).

## Example

```

MD_EQM_ION_T : 0.5 ps
```
