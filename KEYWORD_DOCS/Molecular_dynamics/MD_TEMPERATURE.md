# MD\_TEMPERATURE (.param)

## Keyword type

Real

## Description

This keyword determines the temperature for a molecular dynamics calculation.

This parameter is used only if [MD\_ENSEMBLE](k_md_ensemble_castep.htm) : NVT.

## Default

The default value for this parameter is 300 K.

## Example

```

MD_ENSEMBLE : 275.4 K
```
