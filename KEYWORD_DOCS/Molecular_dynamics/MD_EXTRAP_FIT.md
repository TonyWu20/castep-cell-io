# MD\_EXTRAP\_FIT (.param)

## Keyword type

Logical

## Description

This keyword controls whether or not best fit extrapolation parameters will be calculated after each molecular dynamics step.

If MD\_EXTRAP\_FIT = TRUE, best fit extrapolation parameters will be calculated after each molecular dynamics step.
Otherwise, fixed extrapolation parameters will be used.

## Default

TRUE

## Example

```

MD_EXTRAP_FIT = FALSE
```
