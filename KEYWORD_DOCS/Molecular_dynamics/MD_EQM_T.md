# MD\_EQM\_T (.param)

## Keyword type

Real

## Description

This keyword is used to set the length of the enhanced MD equilibration step (when [MD\_EQM\_METHOD](k_md_eqm_method_castep.htm)
is not set to NONE).

## Default

1 ps

## Example

```

MD_EQM_T : 0.5 ps
```
