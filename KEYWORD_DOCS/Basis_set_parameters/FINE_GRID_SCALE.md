# FINE\_GRID\_SCALE (.param)

## Keyword type

Real

## Description

This keyword determines the maximum size of the g-vectors included in the fine grid relative to the standard grid.

## Default

1 - this results in the fine and standard grids being identical

## Example

```

FINE_GRID_SCALE : 2.0
```
