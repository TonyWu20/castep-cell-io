# LENGTH\_UNIT (.param)

## Keyword type

String

## Description

This keyword specifies the units in which lengths will be reported.

The available units and their associated identifiers are:

| Unit | Identifier |
| --- | --- |
| Bohr | bohr, a0 |
| Meter | m |
| Centimeter | cm |
| Nanometer | nm |
| Å | ang |

## Default

ang

## Example

```

LENGTH_UNIT : bohr
```
