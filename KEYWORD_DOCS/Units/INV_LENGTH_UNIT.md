# INV\_LENGTH\_UNIT (.param)

## Keyword type

String

## Description

This keyword specifies the units in which inverse length will be reported.

The available units and their associated identifiers are:

| Unit | Identifier |
| --- | --- |
| Bohr-1 | 1/ |
| Meter-1 | 1/m |
| Nanometer-1 | 1/nm |
| Å-1 | 1/ang |

## Default

1/ang

## Example

```

INV_LENGTH_UNIT : 1/nm
```
