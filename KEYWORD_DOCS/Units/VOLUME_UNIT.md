# VOLUME\_UNIT (.param)

## Keyword type

String

## Description

This keyword specifies the units in which volume will be reported.

The available units and their associated identifiers are:

| Unit | Identifier |
| --- | --- |
| Bohr3 | bohr\*\*3 |
| Meter3 | m\*\*3 |
| Centimeter3 | cm\*\*3 |
| Nanometer3 | nm\*\*3 |
| Ångstrom3 | ang\*\*3 |

## Default

ang\*\*3

## Example

```

VOLUME_UNIT : nm**3
```
