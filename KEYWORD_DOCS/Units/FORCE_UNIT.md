# FORCE\_UNIT (.param)

## Keyword type

String

## Description

This keyword specifies the units in which force will be reported.

The available units and their associated identifiers are:

| Unit | Identifier |
| --- | --- |
| Hartree per Bohr | hartree/bohr |
| Electron volts per Å | ev/ang |
| Newton | n |

## Default

ev/ang

## Example

```

FORCE_UNIT : n
```
