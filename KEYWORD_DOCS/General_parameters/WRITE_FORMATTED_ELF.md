# WRITE\_FORMATTED\_ELF (.param)

## Keyword type

Logical

## Description

This keyword specifies whether the electron localization function (ELF) is written to an ASCII file called 
seedname.elf\_fmt.

## Default

FALSE

## Example

```

WRITE_FORMATTED_ELF : TRUE
```
