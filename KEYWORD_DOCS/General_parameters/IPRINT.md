# IPRINT (.param)

## Keyword type

Integer

## Description

This keyword controls the level of verbosity of the output. Possible values range from 0, which produces minimal
output, to 3, which corresponds to full debugging output.

## Default

1

## Example

```

IPRINT : 1
```
