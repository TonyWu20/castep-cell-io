# WRITE\_FORMATTED\_POTENTIAL(.param)

## Keyword type

Logical

## Description

This keyword specifies whether the final local potential is written to an ASCII file called seedname.pot\_fmt.

## Default

FALSE

## Example

```

WRITE_FORMATTED_POTENTIAL : TRUE
```
