# CALCULATE\_DENSDIFF (.param)

## Keyword type

Logical

## Description

This keyword controls whether or not a calculation of the density difference with respect to the sum of atomic densities will be performed.

TRUE means that the density difference will be calculated and saved in a binary file
[seed.chdiff](../expcastepfilechdiff.htm).

## Default

FALSE

## Example

```

CALCULATE_DENSDIFF : TRUE
```
