# WRITE\_ORBITALS(.param)

## Keyword type

Logical

## Description

Specifies whether or not to write `seedname.orbitals` binary file in a band structure calculation.

## Default

FALSE unless
`TASK` is BANDSTRUCTURE.

## Example

```

WRITE_ORBITALS : TRUE
```
