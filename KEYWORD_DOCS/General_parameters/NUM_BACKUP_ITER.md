# NUM\_BACKUP\_ITER (.param)

## Keyword type

Integer

## Description

This keyword specifies the number of geometry optimization or molecular dynamics iterations between updates of the backup restart files.

This keyword cannot be used in conjunction with [BACKUP\_INTERVAL](k_backup_iter_castep.htm).

## Default

5

## Example

```

NUM_BACKUP_ITER : 2
```
