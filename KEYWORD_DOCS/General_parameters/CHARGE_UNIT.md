# CHARGE\_UNIT (.param)

## Keyword type

String

## Description

This keyword specifies the units in which charges will be reported.

The available units and their associated identifiers are:

| Unit | Identifier |
| --- | --- |
| Elementary charge | e |
| Coulomb | c |

## Default

e

## Example

```

CHARGE_UNIT : c
```
