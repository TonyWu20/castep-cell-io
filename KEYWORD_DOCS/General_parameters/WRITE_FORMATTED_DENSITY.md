# WRITE\_FORMATTED\_DENSITY(.param)

## Keyword type

Logical

## Description

This keyword specifies whether the final electron density is written to an ASCII file called `seedname.den_fmt`.

## Default

FALSE

## Example

```

WRITE_FORMATTED_DENSITY : TRUE
```
