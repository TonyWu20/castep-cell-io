# TDDFT\_SELECTED\_STATE (.param)

## Keyword type

Integer

## Description

This keyword specified the excited states to use during geometry optimization or molecular dynamics.

## Default

0

## Example

```
TDDFT_SELECTED_STATE : 1
```
