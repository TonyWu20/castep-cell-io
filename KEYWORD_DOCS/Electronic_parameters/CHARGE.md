# CHARGE (.param)

## Keyword type

Real

## Description

This keyword specifies the total charge of the system. It can be used instead of [NELECTRONS](k_nelectrons_castep.htm).

It is not possible to specify the [NELECTRONS](k_nelectrons_castep.htm),
[NUP](k_nup_castep.htm), or [NDOWN](k_ndown_castep.htm) keywords when the CHARGE keyword is specified.

## Default

0

## Example

```

CHARGE : 3
```
