# FIX\_VOL (.cell)

## Keyword type

Logical

## Description

This keyword controls whether or not the volume of the cell remains fixed during relaxation or molecular dynamics. Cell angles and cell lengths may still be varied.

## Default

FALSE

## Example

```

FIX_VOL : TRUE
```
