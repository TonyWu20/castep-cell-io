# FIX\_COM (.cell)

## Keyword type

Logical

## Description

This keyword controls whether or not the center of mass of the ions remains fixed during relaxation or molecular dynamics.

## Default

If [FIX\_ALL\_IONS](k_fix_all_ions_castep.htm) : FALSE then the default value is TRUE.

## Example

```

FIX_COM : TRUE
```
