# FIX\_ALL\_IONS (.cell)

## Keyword type

Logical

## Description

This keyword controls whether or not all of the ionic positions remain fixed during relaxation or molecular dynamics.

## Default

FALSE

## Example

```

FIX_ALL_IONS : TRUE
```
