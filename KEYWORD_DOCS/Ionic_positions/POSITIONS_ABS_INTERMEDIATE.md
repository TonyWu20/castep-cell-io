# POSITIONS\_ABS\_INTERMEDIATE (.cell)

## Keyword type

Block

## Description

This data block contains the positions of the atoms in the intermediate geometry for a transition state search, in absolute coordinates. It
has the same format as [POSITIONS\_ABS](k_positions_abs_castep.htm).
