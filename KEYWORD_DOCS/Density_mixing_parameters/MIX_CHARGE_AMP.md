# MIX\_CHARGE\_AMP (.param)

## Keyword type

Real

## Description

This keyword determines the mixing amplitude for the charge density in the density mixing procedure.

## Default

0.8

## Example

```

MIX_CHARGE_AMP : 0.5
```
