# MIX\_HISTORY\_LENGTH (.param)

## Keyword type

Integer

## Description

This keyword determines the maximum number of densities to store in the history used during the density mixing procedure.

## Default

7

## Example

```

MIX_HISTORY_LENGTH : 5
```
