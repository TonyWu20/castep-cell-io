# MIX\_CHARGE\_GMAX (.param)

## Keyword type

Real

## Description

This keyword determines the maximum g-vector at which the charge density is mixed in the density mixing procedure.

## Default

1.5 Å-1

## Example

```

MIX_CHARGE_GMAX : 0.89 1/ang
```
