# MIXING\_SCHEME (.param)

## Keyword type

String

## Description

This keyword determines which mixing scheme will be used in the density mixing procedure. Available options are:

* Kerker
* Linear
* Broyden
* Pulay

## Default

Broyden

## Example

```

MIXING_SCHEME : Pulay
```
