# POPN\_BOND\_CUTOFF (.param)

## Keyword type

Real

## Description

This keyword controls the maximum distance between two atoms, for which a bond population will be generated, when performing a population
analysis.

## Default

3.0 Å

## Example

```

POPN_BOND_CUTOFF : 2.54 ang
```
