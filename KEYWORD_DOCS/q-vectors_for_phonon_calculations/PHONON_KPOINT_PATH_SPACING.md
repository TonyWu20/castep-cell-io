# PHONON\_KPOINT\_PATH\_SPACING (.cell)

## Keyword type

Real

## Description

This keyword determines the maximum spacing between k-points used for phonon calculations, along the path specified by
[PHONON\_KPOINT\_PATH](k_phonon_kpoint_path_castep.htm).

## Default

0.1 1/ang

## Example

```

PHONON_KPOINT_PATH_SPACING : 0.125
```
