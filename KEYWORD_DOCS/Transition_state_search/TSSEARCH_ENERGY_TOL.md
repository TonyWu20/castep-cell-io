# TSSEARCH\_ENERGY\_TOL (.param)

## Keyword type

Real

## Description

This keyword controls the tolerance for accepting convergence of the maximum energy difference when [TSSEARCH\_METHOD](k_tssearch_method_castep.htm) : NEB.

## Default

0.00002 eV/atom

## Example

```
TSSEARCH_ENERGY_TOL : 0.0005
```
