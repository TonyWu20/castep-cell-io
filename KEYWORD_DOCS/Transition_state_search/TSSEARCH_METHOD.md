# TSSEARCH\_METHOD (.param)

## Keyword type

String

## Description

This keyword controls the search method used to locate a transition state:

* LSTQST - identifies a transition state
* NEB - confirms a transition state using the nudged elastic band method

## Default

LSTQST

## Example

```

TSSEARCH_METHOD : LSTQST
```
