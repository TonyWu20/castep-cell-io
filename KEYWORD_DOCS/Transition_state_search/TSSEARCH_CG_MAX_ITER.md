# TSSEARCH\_CG\_MAX\_ITER (.param)

## Keyword type

Integer

## Description

This keyword determines the maximum number of conjugate gradient steps performed during an LSTQST transition state search.

This parameter must be greater than 0.

## Default

20

## Example

```

TSSEARCH_CG_MAX_ITER : 25
```
