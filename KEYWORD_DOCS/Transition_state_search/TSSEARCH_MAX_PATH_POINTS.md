# TSSEARCH\_MAX\_PATH\_POINTS (.param)

## Keyword type

Integer

## Description

This keyword controls the maximum number of path points to use when [TSSEARCH\_METHOD](k_tssearch_method_castep.htm) : NEB.

## Default

10

## Example

```

TSSEARCH_MAX_PATH_POINTS : 20
```
