# GEOM\_DISP\_TOL (.param)

## Keyword type

Real

## Description

This keyword determines the tolerance for accepting convergence of the ionic displacement during a geometry optimization.

## Default

0.001 Å

## Example

```

GEOM_DISP_TOL : 0.002 ang
```
