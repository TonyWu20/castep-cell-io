# GEOM\_STRESS\_TOL (.param)

## Keyword type

Real

## Description

This keyword controls the tolerance for accepting convergence of the maximum stress component during unit cell optimization.

## Default

0.1 GPa

## Example

```

GEOM_STRESS_TOL : 0.2 GPa
```
