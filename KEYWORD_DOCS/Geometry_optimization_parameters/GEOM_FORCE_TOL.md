# GEOM\_FORCE\_TOL (.param)

## Keyword type

Real

## Description

This keyword controls the tolerance for accepting convergence of the ionic force during a geometry optimization.

## Default

0.05 eV Å-1

## Example

```

GEOM_FORCE_TOL : 0.07 ev/ang
```
