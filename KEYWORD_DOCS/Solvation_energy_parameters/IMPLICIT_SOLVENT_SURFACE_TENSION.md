# IMPLICIT\_SOLVENT\_SURFACE\_TENSION (.param)

## Keyword type

Real

## Description

This keyword specifies the solvent surface tension in the solvation model.

## Default

4.7624e-05 Ha/Bohr2 corresponds to water as a solvent (equivalent to 0.0728 N/m)

## Example

```

IMPLICIT_SOLVENT_SURFACE_TENSION : 0.0738 N/m
```
