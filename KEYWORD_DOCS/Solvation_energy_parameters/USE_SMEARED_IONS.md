# USE\_SMEARED\_IONS (.param)

## Keyword type

Logical

## Description

This keyword specifies whether to use smeared (Gaussian) ion representation in electrostatics with the multi-grid solver. Required for solvation energy calculations.

## Default

FALSE

## Example

```
USE_SMEARED_IONS : TRUE
```
