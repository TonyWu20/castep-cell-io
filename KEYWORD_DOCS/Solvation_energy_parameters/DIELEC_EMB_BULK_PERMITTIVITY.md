# DIELEC\_EMB\_BULK\_PERMITTIVITY (.param)

## Keyword type

Real

## Description

This keyword specifies the relative bulk permittivity for implicit solvation model (dimensionless).

## Default

78.54 (water) if [DIELEC\_EMB\_FUNC\_METHOD](k_dielec_emb_func_method.htm) is set to FG

1.0 (vacuum) otherwise

## Example

```

DIELEC_EMB_BULK_PERMITTIVITY : 32.63
```
