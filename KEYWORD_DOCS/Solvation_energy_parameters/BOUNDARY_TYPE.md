# BOUNDARY\_TYPE (.param)

## Keyword type

String

## Description

This keyword determines appropriate boundary conditions for electrostatic embedding. Allowed values are PERIODIC, OPEN (for embedding or solvation) or ZERO(metallic).

## Default

PERIODIC

## Example

```

BOUNDARY_TYPE : OPEN
```
