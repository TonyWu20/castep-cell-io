# BS\_KPOINT\_PATH\_SPACING (.cell)

## Keyword type

Real

## Description

This keyword determines the maximum spacing between k-points used for non-self consistent band structure calculations, along the path
specified by [BS\_KPOINT\_PATH](k_bs_kpoint_path_castep.htm).

## Default

0.1 1/ang

## Example

```

BS_KPOINT_PATH_SPACING : 0.125
```
