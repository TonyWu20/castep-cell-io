# OPTICS\_KPOINTS\_LIST (.cell)

## Keyword type

Block

## Description

This data block contains a list of k-points at which the Brillouin zone will be sampled during optical matrix element calculations, along
with the associated weights. It has the same format as [KPOINTS\_LIST](k_kpoints_list_castep.htm).
