# KPOINTS\_MP\_SPACING or KPOINT\_MP\_SPACING (.cell)

## Keyword type

Real

## Description

This value specifies the k-point density of a Monkhorst-Pack grid, in units of inverse length.

## Default

0.1 1/Å

## Example

```
kpoints_mp_spacing 0.05 1/ang
```
