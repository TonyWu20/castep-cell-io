```markdown
# MASS_UNIT (.param)

## Keyword type

String

## Description

This keyword specifies the units in which masses will be reported.

The available units and their associated identifiers are:

| Unit             | Identifier |
| ---------------- | ---------- |
| Electron mass    | me         |
| Atomic mass unit | amu        |
| Kilogram         | kg         |
| Gram             | g          |

## Default

amu

## Example

MASS_UNIT : kg
```
